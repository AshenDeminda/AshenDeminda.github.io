PeerLearn is a cross-platform learning app designed to connect students, enable Q&A, and foster collaborative learning. The project leverages modern technologies to ensure scalability, real-time updates, and a smooth user experience.

üöÄ Key Features

  • Email/password sign up & sign in
  • Real-time Q&A: Post doubts and replies with instant updates
  • User profiles & leaderboard: Track contributions and compete
  • Tag system: Categorize questions for easy discovery
  • Responsive design: Works on Android, iOS, Web, Windows, Mac, Linux
