EventHub is a modern web application with a focus on DevOps and deployment automation. The project demonstrates how to build, test, and deploy scalable web services using a fully automated CI/CD pipeline and Infrastructure as Code.

DevOps Tools & Workflow

  • Docker & Docker Compose: Containerized frontend and backend, multiple Compose files for dev/prod environments
  • Jenkins: Automated CI/CD pipeline for building, testing, and deploying containers
  • Terraform: Infrastructure as Code for cloud resource provisioning
  • Ansible: Configuration management and deployment automation

üîÑ Deployment Workflow

  1. Code Commit: Developers push code to the repository
  2. CI/CD Trigger: Jenkins detects changes and triggers the pipeline
  3. Build & Test: Jenkins builds Docker images and runs tests
  4. Provisioning: Terraform provisions or updates infrastructure
  5. Configuration & Deployment: Ansible configures servers and deploys containers
  6. Production Ready: Application is live, reproducible, and scalable
